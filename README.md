# Interview Scheduler Dashboard

## Setup

Install dependencies with `npm install`.

### macOS Monterey

```
nvm install 12.22.7
scheduler % node -v
v12.22.7
scheduler % nvm -v
0.37.2
scheduler % npm -v
8.3.1
```

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```
